## How To Use

```bash
# Clone the repository
$ git clone https://github.com/Dogukanyllmaz/FinalProject---Frontend

# Go to file
$ cd FinalProject---Frontend.git

# Install dependencies
$ npm install

# Start project
$ ng serve -o
```
